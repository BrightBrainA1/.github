## 🛠️ Technical Stack

### Standard Across All Products
- **Frontend:** Next.js 15 (App Router), TypeScript (strict mode)
- **Authentication:** Supabase Auth (self-hosted + managed)
- **Database:** PostgreSQL + Supabase
- **Real-Time:** Supabase Realtime
- **Storage:** Supabase Storage
- **Job Queue:** BullMQ + Redis
- **Observability:** Sentry + Plausible Analytics (self-hosted)
- **Billing:** Stripe
- **Deployment:** Dokploy (self-hosted) + Vercel
